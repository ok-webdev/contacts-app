# Vue 3 + Pinia + Vite + Vuetify

## Description

An app for storing contacts.
Features:

- Create
- Edit
- Delete
- Filter contacts.

Technologies:

- Vue3 (Composition API)
- Pinia
- Vuetify
- Vite

## Deployed

[WATCH HERE](https://contacts-app-pet.netlify.app)

### Setup

```bash
npm install
```

### Develop

```bash
npm run dev
```

### Build

```bash
npm run build
```
